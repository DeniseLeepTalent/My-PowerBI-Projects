# 📊 Power BI Retail & Sales Analysis Project

## 🧾 Overview
This project was completed as part of a **Data Technician Bootcamp**, focusing on building practical skills in **Power BI**. The objective was to transform raw retail and sales data into **clean, structured datasets** and create **interactive reports** that deliver clear, actionable insights through effective data storytelling.

## 🎯 Objectives
- Clean and transform raw retail and sales data  
- Build interactive and user-friendly reports  
- Apply data modelling and DAX calculations  
- Communicate insights through compelling visualisations  

## 🛠️ Tools & Technologies
- **Power BI Desktop**
- **Power Query (ETL processes)**
- **DAX (Data Analysis Expressions)**

## 🔧 Data Preparation & Transformation
- ✅ Used **Power Query** to clean and transform data  
  - Removed errors and duplicates  
  - Standardised formats and data types  
  - Created structured tables ready for analysis  
- ✅ Loaded and modelled data into Power BI for reporting [1](https://b2wcompletetraining057-my.sharepoint.com/personal/denisesorhaindo1_bootcamp_leepgroup_com/_layouts/15/Doc.aspx?sourcedoc=%7B80C3F3A7-6100-44AE-8EC0-9D94EBCF57FD%7D&file=Data_Technician_Workbook_Week_2_May_2026%20final%201%20%281%29.docx&action=default&mobileredirect=true)  

## 📈 Key Features
- ✅ **Interactive Reports**  
  Developed dynamic reports allowing users to explore data through intuitive interactions.

- ✅ **Slicers & Filters**  
  Implemented report-level and visual-level filters to enable:
  - Drill-down analysis  
  - Custom views of sales performance  
  - Region, product, and time-based filtering  

- ✅ **DAX Calculations**  
  Created **calculated columns and measures** to enhance analysis, including:
  - Aggregations (e.g., total sales, averages)  
  - Time-based metrics  
  - Derived KPIs for business insights [1](https://b2wcompletetraining057-my.sharepoint.com/personal/denisesorhaindo1_bootcamp_leepgroup_com/_layouts/15/Doc.aspx?sourcedoc=%7B80C3F3A7-6100-44AE-8EC0-9D94EBCF57FD%7D&file=Data_Technician_Workbook_Week_2_May_2026%20final%201%20%281%29.docx&action=default&mobileredirect=true)  

- ✅ **Variety of Visualisations**  
  Used multiple chart types to effectively present data:
  - Bar charts (product and category comparisons)  
  - Line charts (sales trends over time)  
  - Pie charts (market share and proportions)  
  - Maps (geographical sales distribution)  

## 📊 Data Storytelling Approach
This project emphasises **data storytelling**, ensuring each report communicates a clear narrative. The dashboards were designed to:
- Highlight key retail and sales performance trends  
- Support quick decision-making through visual clarity  
- Provide a logical flow from high-level insights to detailed analysis  
- Enable users to interact with data and uncover their own insights  

## 🔍 Insights & Outcomes
- Identified patterns in sales performance across regions and time periods  
- Enabled comparison of product categories and customer behaviour  
- Improved understanding of business performance through interactive visuals  

## 🚀 Key Skills Demonstrated
- Data cleaning and transformation (Power Query)  
- Data modelling in Power BI  
- DAX (calculated columns and measures)  
- Interactive report design  
- Data visualisation best practices  
- Analytical thinking and insight generation  

## 📌 Conclusion
This project demonstrates the ability to transform raw retail and sales data into **interactive, insightful Power BI reports**. It highlights key technical skills in **data preparation, modelling, and visualisation**, while showcasing the importance of **clear and impactful data storytelling** in business analytics.
